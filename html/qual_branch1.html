perMIT system: Qualifier Hierarchies

This web page allows you to view a hierarchy of qualifiers, such as Funds and Fund Centers, Cost Objects and Profit Centers, etc..

Select a Qualifier type from the drop-down menu. If you know the qualifier code for the branch of the tree you want to view, enter it; otherwise leave it blank and the display will start at the root of the tree. (Qualifier codes must start with the appropriate prefix - see explanation.) Then click the Submit button.

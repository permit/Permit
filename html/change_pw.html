Change Your perMIT system password

Before You Proceed: Get A Secure Web Certificate

To change your Roles password, you must first get secure web certificates from the MIT Certificate Server.

If you already have a username and password for the perMIT system, you can change your password via this secure web interface.

You can use this interface even if you've forgotten your perMIT system password, since the web script uses your Secure Web Certificate to verify who you are.

Remember, your username in the perMIT system must be the same as your Kerberos username, but your password on Roles should be different than your Kerberos password.

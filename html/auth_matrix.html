Show authorizations in a matrix format

This web page allows you to view authorizations within a given category and for a qualifiers of type 'DEPT' as a matrix where the rows are qualifiers and the columns are functions. To use this facility, you need viewing authorizations to see authorizations within the Category you select.

Select a category. Choose format options for the qualifier and for the person. Then click the SUBMIT button.

DLC format: Show DLC name
Show DLC code
